3D TURN BASED GAME PROTOTYPE

APRIL 2020 - NOVEMBER 2020

Project Description

In April 2020, as part of the 3-person team, I started working on a 3D turn-based strategy game prototype to test game concepts and ideas.

In every turn, player can move its party to one of the neighbors fields, using action points. Every field contains a hidden event, the player has to resolve in order to move forward. For every event, player selects characters from their party to participate. Then, event is resolved and the result is calculated based on a randomly generated value and characters' attributes modifiers. Outside of events, player can also use food to heal characters, and use action points to discover neighbor fields to see the event the field is hiding.

For the purpose of this project, I implemented the following things using Unity C# Scripts:

  • Party movement using mouse
  • Characters' health and energy change per turn based on various conditions
  • Randomly generated map
  • Events drawing and resolution logic
  • Basic equipment system with two types of items, permanent and consumable items (affecting only next event)
