BeastEngine + BeastEngineGraphics

AUGUST 2016 - PRESENT

Project Description

The origins of the BeastEngine project go back to 2015 when I started learning games programming. I spent a year creating simple games like 'Tic Tac Toe' or 'Jawbreaker/Bubblets', and I wanted to learn more about the technology behind games in general. That's how I started learning about game engines in 2016 by following a few youtube series which then turned into development of the BeastEngineGraphics.
The initial idea behind it, was to create a 2D game engine which would serve as a framework for a 2D Action-RPG game, but I quickly discoverd that I was more interested in the engine development than the game itself, so I focused entirely on the former.

I was developing the BeastEngineGraphics from August of 2016 to May 2019 in my spare time. I was mostly learning new stuff and trying out different approaches, so eventually it became a sandbox for learning about rendering, and it is more of a rendering library than a game engine. It was written in C++14 and Modern OpenGL, but I also used SDL2, GLM, rapidjson and stb_image.
During the library's development I implemented:

  • Phong lighting
  • OpenGL Framebuffers which allowed me to test some deferred rendering
  • Simple, custom animation system which allowed animating color, position and texture coordinates, based on the std::chrono library
  • Basic materials system
  • 2D normal mapping
I also created the Minesweeper game using the BeastEngineGraphics. You can find the Windows 10 version here.

In 2019, I stopped working on the BeastEngineGraphics and shifted my focus to other game related technologies. I created D3D11 and Vulkan renderers following online tutorials, and created a few simple Unity projects. With new knowledge, at the beginning of 2020, I started working on the BeastEngine, a 2D Action-RPG game engine, successor of the BeastEngineGraphics. It's still in a very early stage, because I was focused mainly on other projects and was only developing this one in my spare time, as a hobby project.
